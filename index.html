Built-in emission factor library

Default factors are illustrative values drawn from IPCC Guidelines, IPCC AR6 (GWP100), and GHG Protocol technical guidance. Each line in Scope 1–3 forms can use one of these defaults or a documented custom factor.

Scope 1 — Direct emissions

Enter activity in the units shown for the selected factor. Emissions (tCO2e) = activity × (kg CO2e per unit) ÷ 1000. Choose a built-in factor or enter your own with a source note.

Scope 2 — Indirect emissions from energy

Report location-based and market-based electricity separately. For a single-method inventory, use one line and leave the other at zero activity. The dashboard sums both lines.

Scope 3 — Value chain emissions

Fifteen categories per the Corporate Value Chain (Scope 3) Standard. Each category has an illustrative default factor; replace with supplier-specific or hybrid factors as needed.
